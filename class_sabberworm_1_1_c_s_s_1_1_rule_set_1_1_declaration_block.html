Detailed Description

Declaration blocks are the parts of a css file which denote the rules belonging to a selector.

Declaration blocks usually appear directly inside a Document or another CSSList (mostly a MediaQuery).

Member Function Documentation

createFontShorthand ( )

Looks for long format CSS font properties (e.g.

font-weight) and tries to convert them into a shorthand CSS font property. At least font-size AND font-family must be present in order to create a shorthand declaration.
